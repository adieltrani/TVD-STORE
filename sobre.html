Sobre a TVD Store

A TVD Store é um projeto de e-commerce totalmente fictício, desenvolvido para demonstrar habilidades em desenvolvimento web front-end, utilizando tecnologias como HTML5, CSS3 e JavaScript puro.

Inspirada no universo da série "The Vampire Diaries", a loja foi criada como um desafio prático para implementar funcionalidades essenciais de uma loja virtual, incluindo:

  • Catálogo de produtos em grade responsiva.
  • Páginas de detalhes dinâmicas para cada produto.
  • Navegação interativa com menu hambúrguer para dispositivos móveis.
  • Formulário de contato funcional com validação de dados em tempo real.

Este projeto não comercializa produtos reais. Seu único objetivo é servir como uma peça de portfólio, mostrando a aplicação de conceitos de design, usabilidade e programação.
